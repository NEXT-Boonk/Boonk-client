24. Scripting API Reference | Project.Model.Modulator

Types:

  • model.ADSRModulator An AHDSR curve modulator applied to a single property of an AutomatableObject.
  • model.AutopitchModulator A modulator which affects the pitch of a Sound across the range of a GameParameters.
  • model.LFOModulator A low frequency oscillator modulator that applies a function to a single property of an AutomatableObject.
  • model.Modulator Base class for objects which additively affect the value of an AutomatableObjecte's property.
  • model.RandomizerModulator A modulator that applies random values to a single property of an AutomatableObject.
  • model.SidechainModulator A modulator that uses the input signal from an external source to apply random values to a single property of an AutomatableObject

model.ADSRModulator

An AHDSR curve modulator applied to a single property of an AutomatableObject.

See Also: AHDSR Modulator, model.AutomatableObject

model.AutopitchModulator

A modulator which affects the pitch of a Sound across the range of a GameParameters.

See Also: Autopitch Modulator, model.Sound, model.GameParameter

model.LFOModulator

A low frequency oscillator modulator that applies a function to a single property of an AutomatableObject.

See Also: LFO Modulator, model.AutomatableObject

model.Modulator

Base class for objects which additively affect the value of an AutomatableObjecte's property.

See Also: Modulator, model.AutomatableObject

model.RandomizerModulator

A modulator that applies random values to a single property of an AutomatableObject.

See Also: Random Modulator, model.AutomatableObject

model.SidechainModulator

See Also: Sidechain Modulator, Sidechain Effect, model.Sidechain, model.AutomatableObject

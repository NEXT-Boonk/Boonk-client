24. Scripting API Reference | Project.Model.Sound

Types:

  • model.CommandSound An instrument that trigeres an API command.
  • model.EventSound An instrument that references an Event to play.
  • model.MultiSound A container instrument that select from a playlist of Sounds.
  • model.PluginSound An instrument that uses an external DSP to generate sound.
  • model.ProgrammerSound An instrument that can be assigned an asset at runtime.
  • model.SilenceSound An instrument that plays nothing for a specified amount of time.
  • model.SingleSound An instrument that plays an asset for a specified amount of time.
  • model.Sound Base class for an instrument.
  • model.SoundScatterer A container instrument that applies 3D attributes to a playlist of Sounds.

Extension Methods:

  • Sound.setFadeInCurve Sets the fade-in curve of the sound to the specified length and curveShape.
  • Sound.setFadeOutCurve Sets the fade-out curve of the sound to the specified length and curveShape.

Sound.setFadeInCurve

Sets the fade-in curve of the sound to the specified length and curveShape.

Sound.setFadeInCurve(
    length,
    curveShape
)
length
number Duration of FadeCurve.
curveShape
Type of FadeCurve to use.

If no fade-in curve exists on the sound, one is created. If length is approaching zero, no curve is added - if the curve has an existing fade-in curve, it is removed. If no curveShape is provided, the default shape is used instead.

Sound.setFadeOutCurve

Sets the fade-out curve of the sound to the specified length and curveShape.

Sound.setFadeInCurve(
    length,
    curveShape
)
length
number Duration of FadeCurve.
curveShape
Type of FadeCurve to use.

If no fade-out curve exists on the sound, one is created. If length is approaching zero, no curve is added - if the curve has an existing fade-out curve, it is removed. If no curveShape is provided, the default shape is used instead.

See Also: Instrument

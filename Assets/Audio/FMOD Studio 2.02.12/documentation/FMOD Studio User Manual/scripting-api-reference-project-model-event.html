24. Scripting API Reference | Project.Model.Event

Types:

  • model.Event An arrangement of sounds that can be played by the game.
  • model.Snapshot An instanceable unit of changes to the project mix.

Extension Methods:

  • Event.addGameParameter Adds a game parameter to the event.
  • Event.addGroupTrack Adds a group track with the given name to the event.
  • Event.addMarkerTrack Adds a marker track to the event.
  • Event.addReturnTrack Adds a return track with the given name to the event.
  • Event.get3DAttributes Get the 3D attributes of an event.
  • Event.getCursorPosition Gets the cursor position.
  • Event.getParameterPresets Get an array of parameters reference by the event.
  • Event.getPath Get the event's unique path identifier.
  • Event.getPlayheadPosition Get the playhead position of a given parameter.
  • Event.is3D Reports whether or not an event is 3D.
  • Event.isOneShot Reports whether or not an event is a oneshot.
  • Event.isPaused Reports whether or not an event is paused.
  • Event.isPlaying Reports whether or not an event is playing.
  • Event.isRecording Reports whether or not an event is recording.
  • Event.isStopping Reports whether or not an event is stopping.
  • Event.keyOff Triggers the keyoff behavior for a sustain point.
  • Event.returnToStart Moves the playback position to the cursor position if called while the event is playing.
  • Event.set3DAttributes Sets the currently auditioned 3D attributes based on an attributes object.
  • Event.setCursorPosition Sets the cursor position to a number.
  • Event.stopImmediate Stops an event immediately, skipping the stopping state.
  • Event.stopNonImmediate Stops an event. The event will enter the stopping state.
  • Event.togglePause Toggle pause for an event.
  • Event.toggleRecording Toggles the recording state for a profiler session.

Event.addGameParameter

Adds a game parameter to the event.

Event.addGameParameter(
    parameterDefinition
)
parameterDefinition
Either be a model.ParameterPreset, a model.GameParameter, or an Object defining the parameter to create:
  • name: The name of the parameter as a string.
  • type: A project.parameterType.
  • min: The minimum value of the parameter range (number).
  • max: The maximum value of the parameter range (number).

Returns the game parameter managed object.

Returns the newly created model.GroupTrack.

Event.get3DAttributes

Get the 3D attributes of an event.

Event.get3DAttributes()

Returns an object representing the currently auditioned 3D attributes:

  • radialDistance: Distance from the listener (number)
  • azimuth: Angle of event in relation to listener (number)
  • height: Vertical position of the event (number)
  • rotation: The horizontal rotation of the event (number)

Event.getParameterPresets

Get an array of parameters reference by the event.

Event.getParameterPresets()

Returns an array of all model.GameParameter objects referenced by the event. This includes any implicitly referenced parameters. The order of the array matches the order displayed in the transport bar.

Event.is3D

Reports whether or not an event is 3D.

Event.is3D()

Returns true if an event is 3D. An event is considered 3D if any of these conditions are met:
- The event has a Spatializer, 3D Object Spatializer, or a 3rd party spatializer on its master track.
- The event contains an automatic parameter that depends on the event's 3D attributes:
- Distance
- Event Cone Angle
- Event Orientation
- Direction
- Elevation
- The event contains any nested events which are 3D.

This function performs calculations that require the Event be fully loaded into memory, this can be slow if triggered on a large number of Events.

Event.isOneShot

Reports whether or not an event is a oneshot.

Event.isOneShot()

Returns true if an event is oneShot. An event is considered oneshot if it is guaranteed to terminate without intervention in bounded time after being started.

This function performs calculations that require the Event be fully loaded into memory, this can be slow if triggered on a large number of Events.

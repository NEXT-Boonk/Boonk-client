24. Scripting API Reference | Project.Workspace

The root workspace * ManagedObject of the project.

Methods:

  • workspace.addEvent Adds an event with the given name to the root folder of the project.
  • workspace.addGameParameter Adds a preset parameter to the project.
  • workspace.addTag Adds a tag with the given name to the project.
  • workspace.createPlugin Creates either a plug-in effect or a plug-in sound.

workspace.addEvent

Adds an event with the given name to the root folder of the project.

workspace.addEvent(
    name,
    withSpatializer
)
name
Name ti give the newly created event.
withSpatializer
Whether to add a spatializer.

Returns the event ManagedObject.
The withSpatializer argument should be true if a spatializer on the master track is desired and false otherwise.

workspace.addGameParameter

Adds a preset parameter to the project.

workspace.addGameParameter(
    parameterDefinition
)
parameterDefinition
parameterDefinition describing the game parameter to add.

Returns the preset parameter ManagedObject.
The properties of the parameterDefinition object match the Event.addGameParameter function.

workspace.addTag

Adds a tag with the given name to the project.

workspace.addTag(
    name
)
name
name of tag to add.

If the tag already exists in the project, the existing tag will be returned instead. Returns the tag managed object.

workspace.createPlugin

Creates either a plug-in effect or a plug-in sound.

workspace.createPlugin(
    identfier
)
identifier
The plug-in to use for the creation. This must match the name of an FMOD_DSP_DESCRIPTION.

Returns the plug-in effect or plug-in sound managed object.

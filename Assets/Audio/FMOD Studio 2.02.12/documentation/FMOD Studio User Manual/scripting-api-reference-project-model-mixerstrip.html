24. Scripting API Reference | Project.Model.MixerStrip

Types:

  • model.EventMixerGroup A group bus within an event.
  • model.EventMixerMaster The master bus of an event.
  • model.EventMixerReturn A return from a return track that can receive signal from events.
  • model.MixerBus Base class for routable buses.
  • model.MixerGroup A MixerBus which accepts other MixerBus inputs.
  • model.MixerInput A MixerBus that allows output from a particular type of model.Event to enter the project-level Mixer.
  • model.MixerMaster The final output bus in a Mixer.
  • model.MixerPort An output to an external audio port.
  • model.MixerReturn A return from a return bus that can receive signal from a MixerSend.
  • model.MixerStrip Base class for elements represented as a strip.
  • model.MixerVCA A MixerStrip which scales the volume of any other MixerStrips assigned to it.

Extension Methods:

  • MixerBus.getInputFormat Get the input format of the bus.
  • MixerBus.getOutputFormat Get the output format of the bus.
  • MixerStrip.getPath Get the MixerStrip's unique path identifier.

model.MixerInput

A MixerBus that allows output from a particular type of model.Event to enter the project-level Mixer.

This bus does not affect the signal, besides mixing together signal from all instances of the associated event type.

See Also: model.MixerBus, model.Event, Mixer Strip
